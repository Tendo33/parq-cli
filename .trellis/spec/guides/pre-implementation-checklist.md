# Pre-Implementation Checklist

- [ ] Which public command, flag, file format, output mode, script, or workflow is affected?
- [ ] Which tests already cover the behavior?
- [ ] Does the change affect automation-facing `plain` or `json` output?
- [ ] Does it affect large-file streaming, overwrite safety, or cleanup on failure?
- [ ] Does README, Chinese README, docs, examples, or release docs need a matching update?
- [ ] Does `.trellis/spec/` need to record a new durable convention?
